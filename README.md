## Getting Started

1. Clone the repository
2. Open `index.html` in a web browser
3. No build process or dependencies required!

## Interactions

- **Anatomy Section**: Click on the Machboos dish image to toggle between complete dish and ingredient anatomy
- **Delivery Tracker**: Click "Simulate Delivery" to watch the order progress through all delivery stages
